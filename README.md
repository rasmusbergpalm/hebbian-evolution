# Hebbian Evolution

Code for the paper "Testing the Genomic Bottleneck Hypothesis in Hebbian Meta-Learning" [arxiv.org/abs/2011.06811](https://arxiv.org/abs/2011.06811)

### Reproducing results

1. `git checkout <revision>`
1. `pip install -r requirements.txt`
1. `python main.py`
